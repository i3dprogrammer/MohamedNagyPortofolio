Description

You can detect your friends location on map, you have ability to set comment or notes on any places that can detected by others, get notification about the near friends of you at any place you are.

App Features

  • Main Process
    • User Account Process
    • Add Friend Process
    • Notification Process
    • Users Search Process
    • Flag Likes Process
  • Flags
    • Process
      • Flags hold Name, description and picture. All flags have 450 meter region to detect when You set this flag on any location on map, anyone enter flag region gets notification about your flag and can read flag name, des, creator and likes on it also users can set like on flags when they are in flag region.
    • Types
      • Public Flags
        • All users can detect your flag
      • Friends Flags
        • Your friends only who can detect your flag
    • Usage
      • Can used as a marketing tool by set offers in flag to notify near users
      • Can used as a comment on place/resturant to notfiy up comming people about this place/res.
      • Can used as a remark for your special place like your home to notfiy near friends to come to visit you
    • Limitation
      • Every user can set only one flag, if user want to create another one then he have to delete the previous one first.
      • when user get notification that there is a nearby flag then he have to go away from flag about 800 meter to active flag notification again.
  • Detection
    • All near friends (250m or less than) can get notifiction that you are closer to them
  • Help Me Mode
    • Used in emergency. When this mode is active then user's friends get notification inwhich this user need help and provide them with user location on map

App Security

  • User can prevent anyone from his friends to detect his location
  • User have ability to open/close for getting notifications about public/friends flags

App API

  • Google Map API
  • Realtime Firebase Database
  • Firebase Database Storage
  • Firebase Authentication
